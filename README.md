# Apple Homepage UI Clone

This project is a responsive clone of the Apple homepage built using React (Vite) and Bootstrap for layout and responsiveness. The application is structured using React functional components, organized into nine reusable components to maintain clean architecture and scalability.

Styling is implemented using a combination of Bootstrap and custom CSS for precise UI control and layout refinement. The design closely replicates Apple’s modern interface style, including typography powered by the Montserrat Google Font and icons integrated using Font Awesome.

The project demonstrates component-based architecture, responsive design principles, and structured frontend development practices using modern React tooling.

## 🛠️ Technologies Used

- ⚛️ React (JavaScript)
- ⚡ Vite
- 🎨 Bootstrap
- 🌐 HTML5
- 🎯 CSS3

## 🚀 Features

- ⚛️ Built with React (Vite) for fast development and optimized performance
- 🧩 Component-Based Architecture using 9 reusable functional components
- 📱 Fully Responsive Design powered by Bootstrap
- 🎨 Custom Styling with CSS for detailed UI control
- 🔤 Google Font Integration (Montserrat) for modern typography
- ⭐ Font Awesome Icons for scalable and clean icon usage
- 🏗️ Structured Project Organization for maintainability and scalability

## Installation

#### Install my-project with npm

```bash
 git clone https://github.com/Kalx6/Apple-Responsive-react.git
 cd Apple-Responsive-react
 npm install
 npm run dev
```

#### Then open:

```
http://localhost:3000
```
